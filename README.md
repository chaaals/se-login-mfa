# Sign up and Log in system with Multifactor Authentication

### Sign up requirements

1. User has to enter the following required fields:

- First name
- Last name
- Username
- Email
- Password

2. The form must have an input validation following these rules:

- Username must be unique
- One email for a user
- Password must have a length of 8 characters

### Log in requirements

1. Use OTP as a form of Multifactor Authentication
2. OTP must be sent to the email of the registered user
